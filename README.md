# GoTorrent - Flashingly fast torrent client

## Installation & Usage

### Quick Start
1. **Grap binary**:
   ```bash
   curl -L https://github.com/xolra0d/gotorrent/releases/latest/download/gotorrent -o gotorrent
   ```

2. **Run the client**:
   ```bash
   chmod +x gotorrent
   ./gotorrent
   ```

### Example Usage
```bash
./gotorrent "magnet_link"
```
